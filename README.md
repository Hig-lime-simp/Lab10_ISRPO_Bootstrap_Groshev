## Описание проекта

Финальное портфолио-сайт, созданный с использованием Bootstrap 5.
Светло-лаймовая цветовая гамма, минималистичный дизайн, адаптивная верстка.

---

## Что было изучено:

- Подключение Bootstrap через CDN
- Система сеток (Grid System)
- Компоненты: Navbar, Cards, Forms, Modal, Accordion
- Утилитные классы
- Адаптивные изображения
- Валидация форм
- Кастомизация стилей

## Структура проекта:

```
Lab10_Bootstrap_Groshev/
├── img/ # Скриншоты
├── custom.css # Кастомные стили (лаймовая тема)
├── experiment.html # Эксперимент с карточкой
├── gallery.html # Галерея с фильтрацией
├── index.html # Главная страница
├── portfolio.html # Финальное портфолио
├── README.md # Описание проекта
├── timeline.css # Стили для временной шкалы
└── timeline.html # Временная шкала достижений
```

---


## Как запустить:

1. Открыть `portfolio.html` в браузере
2. Или использовать Live Server в VS Code

- HTML5
